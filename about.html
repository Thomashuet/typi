About Typi

Typi is a tool to learn a programming language in a web browser without installing anything. It is made by Thomas HUET and the source code is available on GitHub.

Writing tutorials for Typi

Typi tutorials are written in HTML. They can include elements with class question which can themselves contain elements with class test. Elements with class test must contain valid code snippets that evaluate to true if the question is done. See example.html for a complete example.

Distributing a tutorial with Typi

If your tutorial is located at http://example.com/tutorial.html, then use the link http://typi.fr?tuto=http%3A%2F%2Fexample.com%2Ftutorial.html and make sure that you authorize cross domain access to it.
